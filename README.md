## Build instructions:

Dependencies: `clang-format`, `cmake`, `make` and `make`'s dependancies.

Modified from the author's template at https://github.com/ArthurSonzogni/ftxui-starter

```bash
mkdir build
cd build
cmake ..
make -j3
./joss
```
